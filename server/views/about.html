Om Wordle-projektet

Projektöversikt

Wordle-projektet är en enkel webbaserad spel där spelare försöker gissa ett fembokstavsord inom sex försök. Spelet ger återkoppling för varje gissning, vilket indikerar vilka bokstäver som är korrekta och på rätt plats, korrekta men på fel plats, eller inkorrekta.

Funktioner

  • Interaktivt spel med återkoppling för varje gissning
  • Slumpmässig ordgenerering för varje spel
  • Responsiv design för skrivbord och mobila enheter

Teknik som används

Detta projekt är byggt med följande teknologier:

  • HTML, CSS och JavaScript för frontend
  • Node.js och Express för backend
  • Jest för testning

Hur man kör projektet

För att köra projektet lokalt, följ dessa steg:

  1. Klona repositoryn från GitHub.
  2. Navigera till projektmappen och installera beroenden genom att köra npm install.
  3. Starta utvecklingsservern genom att köra npm start.
  4. Öppna din webbläsare och gå till http://localhost:3000 för att spela spelet.
